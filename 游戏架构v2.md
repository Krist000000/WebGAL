# 《路易斯安那之路》v2 架构文档
## 8 章 · Fate 式多路线视觉小说

> 本文件定义游戏的章节结构、路线分叉系统与字数预算。脚本写作须严格遵守本架构。
> 引擎：WebGAL。所有剧本为 `packages/webgal/public/game/scene/*.txt`。

---

## 一、路线系统（Fate / stay night 式）

### 1.1 三线定义

| 路线 | 代号 | 主视角人物 | 主题 | 历史基调 |
|------|------|-----------|------|----------|
| **A 领导者线** | `route_a` | 梅里韦瑟·刘易斯 + 威廉·克拉克 | 测绘、外交、联邦政治压力、个人孤独 | 官方历史 / 架空归隐 |
| **B 边缘者线** | `route_b` | 萨卡加维亚 + 约克（双视角轮换） | 跨文化女性、奴隶制、母性、被历史抹去 | 历史沉默 / 平反 / 自由 |
| **C 基层线** | `route_c` | 约翰·奥德韦 + 汤姆·威拉德（日记体） | 荒野生存、疾病、兄弟情谊、对文明的祛魅 | 平安返乡 / 悲剧 / 创伤 |

### 1.2 分叉时机

- **第一章（共通）**：全玩家共享。圣路易斯集结 → 密苏里河逆流而上。
- **第二章（共通 + 分叉点）**：到达曼丹堡，冬营。第二章**中段**出现「路线选择」——玩家在曼丹堡的立场表态，将按累计好感锁定路线。
- **第三章起**：进入所选人物的独占剧情，直到第八章终章。

> 共通剧情 = 第一章全本 + 第二章前半（到达曼丹堡至分叉选择前）。**不得少于 40,000 字（中文字符）。**

### 1.3 路线进入条件（好感参数）

第一章与第二前段的选择累计三个隐藏参数：

```text
lewis_aff   政治/测绘/联邦权威 类选择
edge_aff     empathy 对约克·萨卡加维亚·印第安视角 类选择
soldier_aff 兄弟情谊/生存/基层视角 类选择
```

第二章分叉选择时，取三者最大值决定路线；若平局，按 `lewis_aff > edge_aff > soldier_aff` 优先级。

分叉语句（写于 `chapter2.txt` 中段）：
```webgal
; 依据好感自动跳转（用 -when 条件）
jumpLabel:to_route_a -when=lewis_aff>=edge_aff && lewis_aff>=soldier_aff;
jumpLabel:to_route_b -when=edge_aff>=soldier_aff;
jumpLabel:to_route_c;
```
或直接给玩家一个明确的「立场选择」`choose`，但内部仍按好感辅助提示。

### 1.4 文件命名

| 文件 | 内容 |
|------|------|
| `chapter1.txt` | 共通第一章 |
| `chapter2.txt` | 共通第二章（含分叉跳转） |
| `route_a_ch3.txt` … `route_a_ch8.txt` | 刘易斯线 第3–8章 |
| `route_b_ch3.txt` … `route_b_ch8.txt` | 边缘者线 第3–8章 |
| `route_c_ch3.txt` … `route_c_ch8.txt` | 士兵线 第3–8章 |
| `end_a_*.txt` / `end_b_*.txt` / `end_c_*.txt` | 各线结局 |

### 共通线

**第一章 · 逆流而上（1804 夏–秋）** 〔共通，约 20,000 字〕

- 冬营中的三方微剧情：刘易斯的政治焦虑 / 约克的尊严 / 威拉德的日记

## 四、分支与 IF 触发点（全局）

| IF | 触发 | 影响 |
|----|------|------|
| IF-1 | 第二章若设计「是否带萨卡加维亚」子选择（可选） | 不带 → 困死雪山悲剧（归入 B 线或独立） |
| IF-2 | 黑脚族冲突中外交信赖/edge 过低 | 冲突升级为国际争端（西班牙/英国介入） |
| IF-3 | 第三章/第五章高风险决策 | 刘易斯提前死亡，历史书写改写 |

> 三线在曼丹堡、落基山、黑脚族三节点须有「互文」：即便进入不同路线，也应通过旁白/日记提及另外两线的关键事件，维持世界观统一。

## 五、写作规范（给脚本作者）

1. **语言**：简体中文，自然口语化对话；旁白沉稳；独白诗意。
2. **每句长度**：对话框单行≤40 字，避免大段。
3. **WebGAL 格式**：
   - 对话 `角色:文本;`
   - 旁白 `:文本;`
   - 立绘 `changeFigure:fig_xxx.png -left -next;`
   - 背景 `changeBg:bg_xxx.png -next;`
   - 选择 `choose:选项A:目标.txt|选项B:目标.txt;`
   - 变量 `setVar:edge_aff=edge_aff+10;`
   - 跳转 `jumpLabel:名;` / `label:名;`
   - 黑屏 `intro:第一行|第二行;`
4. **历史锚点不可动**：购地、共享指挥隐瞒、萨卡加维亚与婴儿、曼丹堡过冬、弗洛伊德唯一死亡、黑脚族冲突刘易斯中枪、1806 凯旋。
5. **每章结尾**用 `changeScene` 或 `jumpLabel` 接到下一章；路线终章用 `end;`。
6. **字数自测**：每文件写完后附 `; STAT: 约 X 字` 注释，便于汇总。
