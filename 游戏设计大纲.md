## 一、世界观与历史地基

### 1.1 必须锁死的真实史实锚点

以下内容是大纲不可违背的「史实地基」，IF 演绎只能在其间填充，不可颠倒基础事实：

| 锚点 | 史实（必须保留） |
|------|------------------|
| **政治发起** | 1803 年杰斐逊向国会秘密申请 2500 美元探索密苏里河，目标：寻找西北水道、开展印第安外交 |
| **购地加成** | 1803 年 5 月 2 日路易斯安那购地案，美国领土翻倍至约 828,000 平方英里；价格存在 **1500 万美元 vs 2700 万美元** 的史料分歧——此为「国家使命」戏剧性来源之一 |
| **核心人物** | 领队梅里韦瑟·刘易斯上尉（杰斐逊秘书），共同指挥威廉·克拉克（实际官方军衔仅少尉，两人对队员隐瞒）；翻译萨卡加维亚（肖松尼人，约 16 岁且怀孕）与其夫夏博诺；克拉克奴隶约克随行；刘易斯纽芬兰犬希曼 |
| **时间线** | 1804-05-14 从圣路易斯附近出发；1804–05 冬驻曼丹堡；1805-11 抵太平洋建科拉特索普堡；1806-03-23 返程，1806-09-23 返圣路易斯 |
| **科学发现** | 识别 178 种新植物、122 种动物；仅军需官弗洛伊德因阑尾炎病逝（唯一死亡者） |
| **冲突危机** | 返程刘易斯分队与黑脚族冲突，杀 2 人；刘易斯意外臀部中枪；西班牙总督曾煽动科曼奇人刺杀未果 |

### 1.2 IF 演绎的合法空间

IF 分支**只允许**改动以下内容，且每个改动必须对应史实留白：

- 某关键人物是否「加入/缺席」（萨卡加维亚若未加入 → 肖松尼马匹交易）
- 局部冲突是否「升级」（黑脚族冲突 → 国际争端）
- 人物生死的「时点」（刘易斯若死于意外而非 1809 年疑云）
- 政策结果的「归属」（约克结局的自由问题）

> **红线**：杰斐逊购地本身、远征最终回返圣路易斯、弗洛伊德之死为唯一死亡、黑脚族冲突中刘易斯臀部中枪——这四条不能动。

## 二、三角色背景板

### 2.1 线索 A · 领导者：梅里韦瑟·刘易斯（Meriwether Lewis）

| 项 | 内容 |
|----|------|
| **身份** | 杰斐逊私人秘书，远征队长（玩家视角 = 队长决策线） |
| **视角核心** | 政治压力 + 科学测绘 + 人格矛盾 |
| **性格弧** | 从「杰斐逊的忠臣」 → 「荒野中的孤独测绘者」 → 「被赞誉包围却愈发沉默的人」 |
| **关键张力** | ① 杰斐逊期望 vs 现实泥沼；② 与克拉克共享指挥却被陆军官僚否认（官方仅给克拉克少尉衔，两人向队员隐瞒）；③ 博物学者的敏锐 vs 对暴力的迟疑；④ 1809 年死亡疑云（IF 线可提前其死亡） |
| **玩家扮演点** | 测绘路线决策、外交取舍、如何分配有限的馈赠、是否执行杰斐逊的秘密指令 |
| **声音形象** | 文本冷静精确，独白中夹杂拉丁文学名与测绘数据；与队员对话时礼貌但疏离 |

### 2.2 线索 B · 边缘者/被压迫者：约克（York）与萨卡加维亚（Sacagawea）双视角轮换

> 采用「双主角轮换制」：同一段剧情从约克与萨卡加维亚两个角度各叙述一次，形成互文。

**约克（York）**

| 项 | 内容 |
|----|------|
| **身份** | 克拉克的奴隶（非自愿随行），远征中承担重劳力却不得奖励 |
| **视角核心** | 奴隶制的日常荒诞 + 边疆自由想象 + 后世被历史抹去 |
| **性格弧** | 从「顺从的奴隶」 → 「在荒野中被印第安人误认为酋长的意外尊严」 → 「归返后身份碎裂的沉默」 |
| **关键张力** | ① 在曼丹堡被误认为「大黑人酋长」获得的短暂尊重 vs 归返圣路易斯后重回奴籍；② 克拉克的相对友善 vs 奴隶制的结构性暴力；③ 其妻子与家人留在路易斯维尔（远征期间他无人可通讯） |
| **IF 触发点** | 玩家选择是否让他在「落基山逃亡」（见结局矩阵） |
| **声音形象** | 对话简短、隐忍，内心独白通过梦境与劳动歌谣呈现 |

**萨卡加维亚（Sacagawea）**

| 项 | 内容 |
|----|------|
| **身份** | 肖松尼人，约 16 岁，被黑脚族掳走后卖予夏博诺，随队翻译；远征初期怀孕，1805-02-11 产子让·巴蒂斯特 |
| **视角核心** | 跨文化女性的翻译困境 + 母性 + 被迫迁徙 |
| **性格弧** | 从「被俘的财产」 → 「队伍的和平通行证」 → 「后世被铸成金币的图腾」 |
| **关键张力** | ① 肖松尼老家近在咫尺却身不由己；② 她的婴儿让队员从「战团」变成「家庭」；③ 夏博诺的笨拙与她的丈夫身份；④ 后世（2000 年）她的形象被铸上 1 美元金币——这与她生前的沉默形成强烈反差 |
| **IF 触发点** | 「若她未加入」 → 肖松尼马匹交易失败、全队困死雪山（见 IF 线） |
| **声音形象** | 翻译时双语切换（肖松尼语/法语/英语），沉默时长于说话，独白中保留大量未翻译的母语词汇 |

### 2.3 线索 C · 基层士兵/自然观察者：约翰·奥德韦（John Ordway）+ 虚构士兵「汤姆·威拉德」

> 采用「日记体双视角」：真实士兵奥德韦的公开日记 + 虚构士兵威拉德的私人日记（两者在同一事件上的叙述差异构成互文）。

| 项 | 内容 |
|----|------|
| **身份** | 军士约翰·奥德韦（真实人物，远征中写日记最勤的士兵之一） + 虚构二等兵汤姆·威拉德（玩家视角） |
| **视角核心** | 兄弟情谊 + 疾病与饥饿 + 对「文明」的祛魅 |
| **性格弧** | 从「志愿入伍的热血青年」 → 「在坏血病与狼群中度日的幸存者」 → 「返乡后无法再适应圣路易斯酒馆的人」 |
| **关键张力** | ① 弗洛伊德之死（唯一死亡）带来的队伍恐惧；② 威士忌配给引发的纪律危机（科林斯偷酒受审）；③ 对岸英国北西公司毛皮商的挑衅；④ 对印第安女性/土地的复杂目光 |
| **玩家扮演点** | 日常资源分配、是否参与偷酒、如何处理与黑脚族的遭遇战 |
| **声音形象** | 口语化、带拼写错误（奥德韦真实日记风格），威拉德的日记则更诗意 |

### 2.4 三线交织节点（互文设计）

| 节点 | 刘易斯线 | 约克/萨卡加维亚线 | 士兵线 |
|------|----------|-------------------|--------|
| **曼丹堡过冬 1804–05** | 外交斡旋 + 杰斐逊书信压力 | 约克被误认为酋长；萨卡加维亚初登场 | 严冬生存 + 第一次坏血病 |
| **翻越落基山 1805-09** | 测绘焦虑（以为是大瀑布捷径） | 萨卡加维亚与肖松尼重逢（IF 触发） | 饥饿低谷 + 星空对话（反节奏叙事） |
| **黑脚族冲突 1806-07** | 分兵决策 + 意外枪伤 | 约克在冲突中的位置 | 冲突现场的第一视角 |
| **归返圣路易斯 1806-09** | 凯旋 + 政治算计 | 约克重回奴籍；萨卡加维亚消失于历史 | 返乡不适 + 创伤 |

## 三、核心机制说明

### 3.1 四项核心玩法（与历史活动对应）

| 机制 | 历史对应 | 玩法形态 | 影响参数 |
|------|----------|----------|----------|
| **河流测绘解谜** | 克拉克手工绘制密苏里河/哥伦比亚河地图 | 玩家拖拽「河道碎片」拼接克拉克地图；拼错会导致后续「迷路」支线 | `测绘精度` |
| **印第安外交礼仪模拟** | 向部落赠送和平奖章、长烟斗、交换礼物 | 对话树中嵌入「礼物选择」QTE；选错礼物降低信赖 | `外交信赖` |
| **野外生存资源分配** | 每日消耗野牛/麋鹿计量，坏血病风险 | 每章结算时分配口粮、决定是否狩猎 | `生存物资` |
| **对话选择与立场** | 刘易斯/克拉克对奴隶制、土地的态度 | 关键剧情选择影响「文化理解」���政治压力」 | `文化理解` `政治压力` |

### 3.2 五项实时参数（决定结局）

```text
测绘精度   ← 河流测绘解谜表现
外交信赖   ← 对肖松尼/黑脚/克拉特索普等部落的外交
生存物资   ← 口粮、弹药、药品储备
文化理解   ← 是否尊重印第安视角（约克/萨卡加维亚线关键）
政治压力   ← 杰斐逊/陆军官僚的满意度（刘易斯线关键）
```

> 结局由上述参数 + 关键 IF 触发点共同演算（见第六章矩阵）。

## 四、章节化分支大纲

### 总结构

```text
序章 · 圣路易斯
  ├─ 第一章 · 逆流而上（1804 夏–秋）
  ├─ 第二章 · 曼丹堡的冬天（1804–05）
  ├─ 第三章 · 大瀑布与落基山（1805 夏–秋）
  ├─ 第四章 · 太平洋与科拉特索普堡（1805 冬）
  ├─ 第五章 · 返程分兵（1806 夏）
  ├─ 第六章 · 黑脚族的枪声（1806 夏）
  └─ 终章 · 圣路易斯 1806-09-23
```

### 各章核心事件与分支点

#### 序章 · 圣路易斯（1804-05）
- **演出**：视差卷轴展示密苏里河宽阔河面，克拉克绘制的地图作为 UI 元素渐显
- **核心事件**：队伍集结、购地消息传来、杰斐逊秘密指令交付
- **分支点**：玩家选择「是否让刘易斯向克拉克坦白共享指挥」（影响政治压力）
- **历史细节**：士兵科林斯偷威士忌受审的伏笔

#### 第一章 · 逆流而上（1804 夏–秋）
- **核心事件**：弗洛伊德病逝（史实唯一死亡）、坏血病初现、首次与苏族遭遇
- **分支点**：如何处理弗洛伊德遗体（影响士兵线士气）
- **机制**：第一次河流测绘解谜（密苏里河弯道）

#### 第二章 · 曼丹堡的冬天（1804–05）
- **核心事件**：与曼丹/希达察部落建交、萨卡加维亚与夏博诺加入、约克被误认为酋长
- **IF 触发点 1**：玩家选择「是否让萨卡加维亚加入」（若选否 → 触发 IF-1 困死雪山线）
- **分支点**：如何处理约克被误认酋长的外交红利（文化理解 ±）
- **演出**：Live2D 立绘微表情——刘易斯收到杰斐逊购地成功信件时的震惊

#### 第三章 · 大瀑布与落基山（1805 夏–秋）
- **核心事件**：大瀑布铁架船漏水弃船窘境、饥饿低谷、萨卡加维亚与肖松尼重逢（她认出首领是其兄）
- **IF 触发点 2**：肖松尼马匹交易（依赖萨卡加维亚是否在队）
- **分支点**：是否分兵（史实分兵，玩家可选择不分 → 改变黑脚族冲突）
- **演出**：多边形背景模拟大瀑布；饥饿低谷插入冗长星空对话（反节奏叙事）

#### 第四章 · 太平洋与科拉特索普堡（1805 冬）
- **核心事件**：抵达太平洋、建堡越冬、与克拉特索普部落互动
- **分支点**：是否严格遵守杰斐逊「不介入部落冲突」指令（政治压力 vs 文化理解）
- **机制**：第二次河流测绘解谜（哥伦比亚河）

#### 第五章 · 返程分兵（1806 夏）
- **核心事件**：分兵两路（刘易斯走玛丽亚斯河，克拉克走黄石河）
- **分支点**：玩家选择跟随刘易斯还是克拉克（决定第六章视角）

#### 第六章 · 黑脚族的枪声（1806 夏）
- **核心事件**：刘易斯分队与黑脚族冲突，杀 2 人；刘易斯意外臀部中枪
- **IF 触发点 3**：冲突是否升级为大规模战争（外交信赖低 + 文化理解低 → IF-2 国际争端线）
- **分支点**：冲突中玩家扮演的角色决策（是否开枪、是否追击）

#### 终章 · 圣路易斯 1806-09-23
- **核心事件**：凯旋、国会嘉奖、各人物结局结算
- **结局矩阵**：由五项参数 + 三个 IF 触发点决定（见第六章）

## 五、IF 线与触发点

| IF 编号 | 触发点 | 条件 | IF 剧情走向 |
|---------|--------|------|-------------|
| **IF-1** | 萨卡加维亚未加入 | 第二章选择「不带她」 | 肖松尼马匹交易失败 → 全队困死落基山雪山 → 悲剧结局（仅士兵线可触发） |
| **IF-2** | 黑脚族冲突升级 | 外交信赖 < 30 且文化理解 < 40 | 冲突升级为大规模战争 → 西班牙/英国介入 → 国际争端线 |
| **IF-3** | 刘易斯提前死亡 | 第三章/第五章选择高风险决策 | 刘易斯死于狩猎意外（而非 1809 年疑云）→ 克拉克/士兵带回地图 → 改变历史书写 |

> **IF 设计原则**：每个 IF 必须让玩家在事后看到「历史本可以不同」的震撼，而非纯粹为了多结局。

| 结局 | 条件 | 描述 |
|------|------|------|

| 结局 | 条件 | 描述 |
|------|------|------|
| **C-1 沉默的母亲（历史���）** | 文化理解中 | 史实结局：1812 年病逝（或 1884 年死于肖松尼保留地，两种说法），生前未被铭记 |

### 6.4 士兵线结局（≥2 个）

| 结局 | 条件 | 描述 |
|------|------|------|
| **D-1 平安返乡（历史向）** | 生存物资高 | 史实结局：多数士兵平安返乡，奥德韦继续写日记至 1817 年 |

### 7.1 关键演出镜头清单

### 7.2 反节奏叙事设计

- **原则**：在「探索爽感」最强处插入「冗长对话/沉默」，打破玩家预期
- **具体节点**：
  - 第三章翻越落基山饥饿低谷：插入 10 分钟星空对话（刘易斯与克拉克谈杰斐逊、谈死亡、谈地图空白）
  - 第六章黑脚族冲突后：插入 5 分钟河边沉默（只有水声与呼吸）
- **目的**：让玩家在「赢」的时刻感到「代价」
