# 《路易斯安那之路》美术提示词包
## 日漫精美风格 × 1804 年美国边疆文化考据

> 用途：交给美术 agent / 画师，生成背景图、人物立绘、剧情 CG（故事插画）。
> 每一类给出「正向提示词（英文，适配 Midjourney / SDXL / Niji）」「中文说明」「文化细节清单」「构图要点」。
> **一致性铁律**：所有图必须同时具备「日漫美学」与「1804 年美国西北边疆的物质文化」，二者缺一不可。

---

## 〇、全局风格与负向提示

### 整体美术方向
- **画风**：精致日漫（参考 ufotable / 京都动画的厚涂光影 + PA.WORKS 的自然景），线条干净，色彩电影感，柔光与体积光。
- **时代质感**：在动漫美型基础上，所有器物、服饰、建筑必须历史准确（见各条文化清单），不做现代化简化。
- **色调**：随章节变化——共通线暖金河川；曼丹堡冷蓝雪原；落基山青灰；太平洋橙紫落日；黑脚族冲突暗红火光。

### 全局正向前缀（每张图必带）
```text
masterpiece, best quality, anime style, cinematic lighting,
historical accuracy, 1804 American frontier, intricate detail,
soft volumetric light, painterly, 8k
```

### 全局负向提示（每张图必带）
```text
modern clothing, smartphone, neon, sci-fi, cyberpunk,
anachronism, plastic, concrete skyscraper, car, electricity,
off-shoulder anime costume, exaggerated weapon, floating UI,
3d render, lowres, bad anatomy, extra limbs, watermark
```

## 四、一致性规范（交给美术 agent 的硬性要求）

1. **服饰零穿帮**：1804 年前不得出现圆顶硬礼帽（top hat, 1850+）、西装革履、拉链、纽扣牛仔（1873+）。扣子用骨/木/铜。
2. **发型**：男性短发或束尾，不得出现现代发型；萨卡加维亚深辫；杰斐逊红粉假发束尾。
3. **建筑**：圣路易斯=法式木骨泥墙；曼丹=圆形土屋；太平洋=原木堡；不得出现维多利亚砖楼。
4. **交通**：仅龙骨船/皮罗格/马/独木舟/步行；无车轮载具（除铁架搬运车）。
5. **武器**：1795 式步枪、燧发枪、战斧、和平奖章；无左轮（1836+）、无来复线后装枪。
6. **光影统一**：每章定一个主色温（见背景条），立绘与 CG 须匹配该章色温。
7. **角色一致性**：同一角色在所有图中，发型/服饰/五官须完全一致（建议锁定 seed 或参考图）。
